# قابلیت‌های Carousel

## نمایش اسلایدی عکس‌ها و ویدیوها

اکنون بلاگ‌ها و نمونه‌کارها با یک carousel حرفه‌ای برای نمایش عکس‌ها و ویدیوهای متعدد آپگریید شده‌اند.

### قابلیت‌های Carousel:

#### 1. نمایش اندازه واقعی
- عکس‌ها و ویدیوها با اندازه‌های مختلف بدون تحریف نمایش داده می‌شوند
- استفاده از `object-fit: contain` برای حفظ نسبت طول و عرض
- حداکثر ارتفاع 500px برای نمایش بهتر

#### 2. کنترل‌های متنوع
- **دکمه‌های پیشرو/عقبرو**: برای حرکت سریع بین عکس‌ها
- **نقاط ناوبری (Dots)**: برای حرکت مستقیم به هر عکس
- **شمارنده**: نمایش وضعیت فعلی (مثل 1/5)

#### 3. کنترل‌های کیبوردی
- **فلش راست**: رفتن به اسلاید بعدی
- **فلش چپ**: بازگشت به اسلاید قبلی

#### 4. حرکت Swipe (موبایل)
- حرکت به راست: اسلاید بعدی
- حرکت به چپ: اسلاید قبلی

#### 5. ویدیو embedded
- پشتیبانی YouTube و Vimeo در بلاگ
- ویدیوهای آپلود شده مستقیم در carousel

### استفاده:

#### در بلاگ:
1. ویرایش بلاگ → عکس/ویدیو بارگذاری کنید
2. لینک ویدیو خارجی (YouTube/Vimeo) را اضافه کنید
3. عکس‌ها و ویدیوها خودکار نمایش داده می‌شوند

#### در نمونه‌کارها:
1. ویرایش پروژه → عکس/ویدیوهای متعدد بارگذاری کنید
2. لینک‌های پروژه اضافی اضافه کنید
3. گالری اسلایدی خودکار ایجاد می‌شود

### مثال دادهای JSON (media_files):
```json
[
    "/uploads/blog/507c4d1a1b2c3.jpg",
    "/uploads/blog/607c4d2a2b3c4.mp4",
    "/uploads/blog/707c4d3a3b4c5.png"
]
```

### مثال دادهای JSON (project_links):
```json
[
    {"title": "دمو زنده", "url": "https://example.com"},
    {"title": "GitHub", "url": "https://github.com/..."},
    {"title": "مستندات", "url": "https://docs.example.com"}
]
```

## نکات مهم:

1. تمام فرمت‌های تصویری معروف پشتیبانی می‌شوند (JPG, PNG, GIF)
2. ویدیوهای معروف پشتیبانی می‌شوند (MP4, WebM, MOV)
3. اندازه عکس تاثیری بر نمایش ندارد - همه بهینه می‌شوند
4. Carousel خودکار برای یک عکس/ویدیو دکمه‌ها را مخفی می‌کند
5. کلیدهای کیبوردی فقط هنگام تمرکز بر carousel فعال هستند
